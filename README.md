# AI × Web3 School - 个人学习仓库

这是我在 [AI × Web3 School](https://aiweb3.school/zh/handbook/) 的个人学习记录和 proof-of-work 仓库。

## 关于 AI × Web3 School

AI × Web3 School 是一个面向 builders 的开源学习计划，由 LXDAO 与 ETHPanda 共同发起。

- **Handbook**：https://aiweb3.school/zh/handbook/
- **WCB 课程页面**：https://web3career.build/zh/programs/AI-Web3-School
- **GitHub**：https://github.com/lxdao-official/aiweb3school

## 学员画像

- **AI 基础**：有基础（了解概念，使用过 AI 工具）
- **Web3 基础**：有基础（了解基本概念）
- **编程能力**：会基础编程（C++）
- **目标方向**：开发 + Hackathon 项目 + 内容运营
- **每日投入时间**：1-2 小时
