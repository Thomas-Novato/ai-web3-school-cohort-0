## 学习内容

### Hermes Agent 实践

今天主要学习了 Hermes Agent 的使用，这是一个智能 AI 助手，可以帮助完成各种任务。

**核心功能体验**：
- 代码编写和调试

- 任务规划和执行

**遇到的技术问题**：
- ⚠️ **Gateway 限制问题**：笔记本关机重启后，gateway 会有限制，需要重启服务才能继续使用 bot
- 这个问题影响了使用的连续性，需要找到解决方案

### Prompt 工程的实践思考

通过今天的使用，对 Prompt 有了新的理解：

**关键发现**：
- 与其自己费力写完美的 prompt，不如直接把想法告诉 AI，让 AI 帮忙优化 prompt
- 这是一种"元 prompt"的思路：用 AI 来生成更好的 prompt
- 实际效果比自己琢磨半天要好

**实践模式**：
```
我的想法 → AI 优化 → 更好的 prompt → 更好的结果
```

这种方式更符合实际工作流程，也更高效。

### 理论学习

继续巩固昨天学习的 LLM 章节内容：
- LLM 是推理层，不是真相源
- 需要外部验证来确保准确性
- 在 AI × Web3 中，链上数据是真相源，LLM 负责理解和推理
