# 2026-05-26 打卡草稿

## 今日完成

### 理论学习
- ✅ **MCP（Model Context Protocol）** — 完成了 MCP 基本概念的学习
  - MCP 三层架构：Model → MCP Client → MCP Server
  - 能力边界管理：Tool Schema 决定调用正确性，Permission 决定调用深度
  - AI × Web3 场景：MCP 作为 Agent 与链上工具的接口层
  - 安全实践：从只读开始，逐步增加权限控制、用户确认、审计机制

### 关键收获
- MCP 的核心不是连接能力，而是能力边界管理
- MCP 本身不是安全方案，最终依赖钱包授权、交易模拟、用户签名、审计系统
- 与课程其他模块关联紧密：Agent、Context 可信度分层、Web3 Tool Use、AI Security
