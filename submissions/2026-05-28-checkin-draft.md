# 2026-05-28 WCB 打卡草稿

## 今日完成内容

**学习章节**: 密码学（Cryptography）— Web3 基础
**Handbook 链接**: https://aiweb3.school/zh/handbook/web3/cryptography/

**核心收获：**

1. **私钥决定控制权** — 私钥丢失=失权，泄漏=被盗。钱包、助记词、多签都是围绕私钥管理
2. **签名不等于登录** — 签名是对具体消息/交易的授权证明，不是简单的"确认"
3. **哈希不是加密** — 哈希是数据指纹，不可逆，用于验证一致性
4. **Merkle Tree** — 用 Root + Proof 高效验证数据归属，空投和轻客户端常用
5. **AI × Web3 边界** — AI 可帮用户解释交易和签名，但不应该保管私钥或索要助记词

**学习时长**: ~1 小时
